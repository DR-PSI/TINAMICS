// FAULT & ALARM GUIDE

คู่มือ Fault & Alarm G120 / G120X

Fault และ Alarm codes ที่พบบ่อยในงาน Pump/Fan Applications พร้อม Cause และ Corrective Action

F30001 / F07801
Overcurrent / Motor Overcurrent
กระแสเกินขีดจำกัด — Ramp-up สั้นเกิน, Short circuit, Mechanical Jam
REMEDY

เพิ่ม P1120 (Ramp-up time) · Megger test สายและ Motor · ตรวจ Motor Data · หมุน Shaft ด้วยมือ

F30002
DC Link Overvoltage
Deceleration เร็วเกิน Motor ป้อนพลังงานกลับ DC Bus — Supply Voltage สูงเกิน
REMEDY

เพิ่ม P1121 (Ramp-down time) · Enable P1240 Vdc_max Controller · วัด Input Voltage L1,L2,L3

F30003
DC Link Undervoltage
ไฟฟ้าดับ/ตก (Brownout) · Fuse ขาด · Breaker ตัด · Phase หาย
REMEDY

วัด L1,L2,L3 ครบ 3 Phase · ตรวจ Contactor และ Breaker · Configure P1210 Auto-Restart

F30021
Ground Fault
Ground fault ที่สายไฟ Output หรือ Motor Winding
REMEDY

Megger test สาย Output U,V,W · ตรวจ Motor Winding Insulation · ตรวจ CT

F30024
Overtemperature Thermal Model
โหลดเกิน, ระบบระบายความร้อนพัง, อุณหภูมิแวดล้อมสูง
REMEDY

ตรวจ Fan หมุนอิสระ · ลด Pulse Frequency · ตรวจ Ambient Temperature

F07902
Motor Stalled / Motor Blocked
Load หนักเกิน Motor หมุนไม่ได้ — Motor ID ยังไม่ได้ทำ
REMEDY

หมุน Shaft ด้วยมือ · Run Motor ID (p1900=1) · ตั้ง Current Limit P0640

A05000
Overtemperature Heat Sink
อุณหภูมิ Heat Sink ถึง Alarm threshold — จะ Fault F30004 ถ้าเพิ่มอีก 5K
REMEDY

ตรวจ Ambient Temperature · ตรวจ Load Duty Cycle · ตรวจระบบ Cooling

A07805
Power Unit Overload I²t
I²t Overload เกิน Alarm threshold P0294
REMEDY

ลด Continuous Load · ปรับ Load Duty Cycle · ตรวจ Motor/Drive current rating

A07012
Motor Over Temperature
Motor ร้อนเกิน — Overload นาน, Cooling ตัน, Sensor ผิดพลาด
REMEDY

วัดอุณหภูมิ Motor ด้วย IR Thermometer · ทำความสะอาด Cooling Fin · ตรวจ P0305
